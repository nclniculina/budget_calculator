# Budgetrechner

Ein einfacher Budgetrechner, um Einnahmen und Ausgaben zu verfolgen und den Kontostand zu berechnen. 
Das Projekt zeigt grundlegende HTML-, CSS- und JavaScript-Fähigkeiten und eignet sich gut für ein Portfolio.

## Technologien 
- HTML
- CSS
- JavaScript

## Funktionen 
- Einnahmen hinzufügen 
- Ausgaben hinzufügen 
- Gesamtsummen für Einnahmen und Ausgaben berechnen 
- Kontostand anzeigen 
- Transaktionsverlauf speichern und anzeigen 
- Daten im LocalStorage sichern, sodass sie beim Neuladen erhalten bleiben

## Nutzung 
1. Alle Dateien (index.html, style.css, script.js) im selben Ordner speichern. 
2. index.html in einem Browser öffnen. 
3. Einnahmen und Ausgaben eintragen, auf die Buttons klicken. 
4. Gesamtsummen und Transaktionsverlauf werden automatisch berechnet und angezeigt.

## Struktur des Projekts
Budgetrechner/
├─ index.html # HTML-Struktur der Seite
├─ style.css # Styling für Layout, Farben, Buttons
├─ script.js # JavaScript-Logik für Berechnungen und LocalStorage
└─ README.md # Projektbeschreibung
